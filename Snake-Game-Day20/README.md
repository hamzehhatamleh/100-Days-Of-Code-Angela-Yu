# Day 20: The Snake Game (Part 1 & 2)

An implementation of the classic arcade Snake Game built using Python's object-oriented programming fundamentals and the `turtle` graphics library.

## Core Concepts Practiced:
* **Object-Oriented Programming (OOP):** Segmented game logic into modular classes (`SnakeBody`, `Food`, and `Score`) for clean architecture.
* **Screen Animation & Control:** Leveraged `screen.tracer(0)` and `screen.update()` to enable smooth animation updates instead of disjointed movement.
* **Event Listening:** Implemented key bindings (`screen.listen()`) to handle real-time user steering inputs smoothly.
* **Collision Detection:** Programmed logic to detect when the snake touches the food or impacts borders/its own tail.
