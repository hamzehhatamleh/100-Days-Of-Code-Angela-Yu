# Python Pong Game

A classic 2-player Pong game built using Python's native `turtle` graphics library. 

## How to Play
* **Left Player (W/S):** Move paddle Up and Down.
* **Right Player (Arrow Keys):** Move paddle Up and Down.

## Features
* Smooth paddle collision detection.
* Dynamic score tracking with an authentic retro center line.
* Disconnects animation delay for instant rendering.

## How to Run
1. Make sure you have Python 3 installed.
2. Clone this repository.
3. Run `python main.py` in your terminal.
