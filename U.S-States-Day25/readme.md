# U.S. States Game

A fun, interactive geographical quiz built using Python, Turtle Graphics, and Pandas.

## How to Play
1. A popup box will appear asking you to type a US State.
2. If your guess is correct, the state's name will appear on the map at its exact location.
3. The title bar tracks your current score out of 50.
4. Type `Exit` or close the window to end the game early.

## Installation
Make sure you have your 'blank_states_img.gif' and '50_states.csv' files in the same folder as the script.

```bash
# Clone or download this repository, open your terminal in the folder and run:
pip install -r requirements.txt
python main.py
```
